# Contributing to Oxira

Thank you for your interest in contributing.

## Development setup

```bash
git clone https://github.com/your-username/oxira.git
cd oxira
npm install
cp .env.example .env
# Add your BRAVE_SEARCH_API_KEY to .env
```

## Making changes

1. Create a branch for your change
2. Run `npm run build` and `npm test` to ensure everything passes
3. Run `npm run lint` before submitting
